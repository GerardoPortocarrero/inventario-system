# Sistema de Inventario

Este proyecto tiene como objetivo sentar las bases (arquitectura, base de datos, y flujos de usuario) para un sistema de gestión de inventario avanzado.

## Propósito

El sistema permitirá un control granular del stock de productos a través de múltiples estados y jerarquías físicas, facilitando las operaciones de almacén, la labor de los preventistas y la reportería para sistemas externos como SAP o BASIS.
